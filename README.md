@turbotearsbot

Что хочу сделать из бота, а потом телеговское приложение, которые:

 1. Отслеживает сезон Формулы-1
– список всех гонок сезона,
– практики, квалификации и гонки,
– даты, время старта.

 2. Показывает актуальную статистику
– все пилоты,
– их текущие очки в личном зачёте,
– все команды,
– их очки в Кубке конструкторов.

 3. Умеет следить за любимыми пилотами/командами
– пользователь выбирает, за кем следить.

 4. Отправляет уведомления в Telegram
– напоминание перед стартом гонки,
– уведомление после гонки: место пилота, обновлённые очки.

 5. Работает полностью автоматически
– получает данные из API Формулы-1,
– обновляет результаты в реальном времени или после гонки.
